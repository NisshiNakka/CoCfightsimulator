## サービス概要

このアプリは、TRPG(*1),特にCoCにおいての戦闘のシミュレーションを行うことができるアプリです。調整したデータはそのままココフォリア(*2)に出力することが可能であり、テストをしたデータをスムーズに本番で使うことができます。

(* TRPG ... テーブルトークRPGの略称。ダイスなどを使用して進めるアナログゲーム )
(* ココフォリア ... TRPGを行うための環境を作ることができる現在主流のサイト。)

## このサービスへの思い・作りたい理由

1.自分や友人が実際にTRPGのシナリオを作成していた時に、ゲーム内の戦闘バランス調整を行う事がありました。
その際、ステータスを決めて、攻撃の成否・回避の成否・ダメージの値などの判定を一回一回ダイスを振って行う（ダイスを振り結果を見る）
といったテスト手順を行う必要があり、時間がかかり面倒に感じていました。
そこで、ダイスを用いた判定を自動で行うサービスがあれば、テスト時間の短縮につながり、
結果として十分なテスト回数を確保して完成度を高めたり、テスト時間を短縮して生まれた時間を他の作業に充てることができるのではないかと考え、本アプリを考案しました。


2.創作を行うことが好きで、自分自身動画やTRPGというゲームのシナリオなどを制作する機会が多くありました。
その際、動画や音声、画像の編集ソフトや執筆を行いやすくするwebサービス、複雑な計算式や暗号を解いたり作成することができるツールなど、様々なツールに助けられてきました。
こうしたツールを使って創作活動を行っていくうちに、何かを創ることは楽しい反面ハードルが高いということを感じるようになり、「創ること」による楽しさをもっと体験しやすくしたいと考えるようになりました。
そこで、「創作をもっとしやすく」することができるサービスの一つとして、複雑・面倒な処理を短時間で行えるツールを考えつきました。

ユーザー層　TRPGのシナリオを作る人、シナリオを人数などに合わせ調整する人
理由1.　何かを作ることに関してハードルを下げることができるようなサービスを作りたかったから

## サービスの利用イメージ

ユーザーはこのサービスにアクセスし、敵・味方のステータスなどの値を入力します。
その後、条件を決めて、実行ボタンを押すことで結果が表示されます。
これにより、自分が作ったシナリオの戦闘バランスを考え、テストを行うことが簡単になります。

・現在考えているロジック
1.ユーザーが敵・味方のステータスを入力（HP/攻撃技能/回避値/ダメージ値）
2.入力されたデータを、攻撃/回避行動の成功/失敗値に当てはめる。
3.敵味方各一回ずつ攻撃判定を行う。（各判定の成否はダイスで出た数値によって決定される。）
4.攻撃が成功した場合、ダメージを算出し、ダメージ値分HPを減らす。
5.3〜４を敵味方いずれかのHPがゼロになるまで繰り返す。
6.5の繰り返し処理が終わった段階でユーザーに結果を表示する。

## サービスの差別化ポイント・推しポイント

このサービスのポイント
1.独自性が高い　他に代替できそうなアプリはない
2.ココフォリアの場合、「現在考えているロジック」の手順3〜５のすべての判定を一回一回手入力してテストを行わなければならない。
弊アプリの場合、ステータスや条件を決めたらあとは１クリックで結果がわかるようになり、テスト時間が短縮される。
3.（本リリース後の実装予定機能）
テストを行ったあと、ユーザーが作成した敵・味方のステータスをスムーズに本番環境へ移すことができる。
