## サービス概要

このアプリは、TRPG(*1),特にCoCにおいての戦闘のシミュレーションを行うことができるアプリです。<br>
調整したデータはそのままココフォリア(*2)に出力することが可能であり、テストをしたデータをスムーズに本番で使うことができます。<br>

(* TRPG ... テーブルトークRPGの略称。ダイスなどを使用して進めるアナログゲーム )<br>
(* ココフォリア ... TRPGを行うための環境を作ることができる現在主流のサイト。)

## このサービスへの思い・作りたい理由

1.自分や友人が実際にTRPGのシナリオを作成していた時に、ゲーム内の戦闘バランス調整を行う事がありました。
その際、ステータスを決めて、攻撃の成否・回避の成否・ダメージの値などの判定を一回一回ダイスを振って行う（ダイスを振り結果を見る）
といったテスト手順を行う必要があり、時間がかかり面倒に感じていました。
そこで、ダイスを用いた判定を自動で行うサービスがあれば、テスト時間の短縮につながり、
結果として十分なテスト回数を確保して完成度を高めたり、テスト時間を短縮して生まれた時間を他の作業に充てることができるのではないかと考え、本アプリを考案しました。


私は創作を行うことが好きで、自分自身動画やTRPGというゲームのシナリオなどを制作する機会が多くありました。<br>
その際、動画や音声、画像の編集ソフトや執筆を行いやすくするwebサービス、複雑な計算式や暗号を解いたり作成することができるツールなど、様々なツールに助けられてきました。<br>
こうしたツールを使って創作活動を行っていくうちに、何かを創ることは楽しい反面ハードルが高いということを感じるようになり、「創ること」による楽しさをもっと体験しやすくしたいと考えるようになりました。<br>
そこで、「創作をもっとしやすく」することができるサービスの一つとして、複雑・面倒な処理を短時間で行えるツールを考えつきました。<br>




## サービスの利用イメージ

1.ユーザーはこのサービスにアクセスし、キャラクターを作成します。<br>
2.その後、シミュレーションページにて作成したキャラクターを選択します。<br>
3.実行ボタンを押すことで結果が表示されます。<br>
4.選択中のキャラクターのパラメーターも、「キャラクター編集」にて簡単に変更することができます。<br>

5.ボタンひとつで即座に結果がわかるので、簡単に何度もシミュレーションを行うことができます。<br>
これにより、自分が作ったシナリオの戦闘バランスを考え、テストを行うことが簡単になります。<br>
